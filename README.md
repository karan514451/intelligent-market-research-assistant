
# Intelligent Market Research Assistant

## Overview

The Intelligent Market Research Assistant is an Agentic AI-powered business intelligence platform built using LangFlow, IBM watsonx.ai, IBM Granite Models, RAG, and ChromaDB.

The system automatically collects market information from multiple sources, analyzes competitor activities, tracks customer sentiment, predicts market trends, and generates executive reports for business decision-making.

## Features

* Real-time Market Intelligence
* Competitor Analysis
* Customer Sentiment Analysis
* Trend Forecasting
* Executive Reporting
* RAG-based Knowledge Retrieval
* Interactive Dashboard
* Multi-Agent Architecture

## Tech Stack

* LangFlow
* IBM watsonx.ai
* IBM Granite 4.0-8B-Instruct
* ChromaDB
* Python
* Streamlit
* IBM Cloud
